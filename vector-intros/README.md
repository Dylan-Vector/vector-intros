# Vector Candidate Intro Generator

Precision hiring for AI & Infrastructure companies.

## Setup

1. Add your Anthropic API key as an environment variable in Vercel:
   - Key: `ANTHROPIC_API_KEY`
   - Value: your `sk-ant-...` key

2. Deploy to Vercel by connecting this GitHub repo.

## Usage

1. Enter client name, role title, role type
2. Paste the call transcript
3. Hit Generate
4. Cmd+P to save as PDF
